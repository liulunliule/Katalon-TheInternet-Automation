# Katalon Automation Practice

## 📌 Overview

This project demonstrates automation testing using Katalon Studio with:

* Data-Driven Testing (Excel)
* Custom Keywords (Reusable functions)
* Basic UI automation on The Internet website

## 🧪 Test Scenarios

* Login with valid credentials → Success
* Login with empty password → Error
* Login with wrong username → Error
* Login with empty fields → Error

## 🛠 Technologies

* Katalon Studio
* Groovy
  
## 📂 Project Structure

* Test Cases: Contains all test scenarios
* Object Repository: UI elements
* Data Files: Excel test data
* Keywords: Custom reusable functions

## 🚀 How to Run

### Run via Test Suite (Recommended)

1. Open project in Katalon Studio
2. Go to **Test Suites**
3. Select: `Regression_Suite`
4. Click **Run**

### Included Test Cases

* TC01_Login_Failed_With_Empty_Password
* TC02_Login_DataDriven
* TC03_Checkboxes
* TC04_Dropdown
